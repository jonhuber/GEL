# GEL3 Website

A Keystone + NextJS website.

## Getting started

1. rename `.env.example` to `.env`
2. replace the `DATABASE_URL` variable with your own string. See the [documentation on Knex adapter](https://v5.keystonejs.com/keystone-alpha/adapter-knex/) for more details
3. Run `yarn` to install dependencies
4. `yarn start` to start the server
